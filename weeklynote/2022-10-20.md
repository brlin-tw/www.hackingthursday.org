在 `/linux/drivers/acpi/processor_idle.c`:

```
/*
 * Callers should disable interrupts before the call and enable
 * interrupts after return.
 */
static void __cpuidle acpi_safe_halt(void)
{
        if (!tif_need_resched()) {
                safe_halt();
                local_irq_disable();
        }
}

```

這一段是 CPU halt 的行為，但是不保證 CSTATE 的狀態
