# mosky:

python decorator 的原理

    >>> def d(f):
    ...     def product():
    ...             return f()*3
    ...     return product
    ... 
    >>> def f():
    ...     return 'x'
    ... 
    >>> new_f = d(f)
    >>> f()
    'x'
    >>> new_f()
    'xxx'
    >>> @d
    ... def g():
    ...     return 'y'
    ... 
    >>> g()
    'yyy'
    >>>
