# Mat:

一般在批次作業中，如果需要修改設定檔的內容時，大多是用 sed, awk. 
不過因為對 vim 的指令較熟，於是就想說有沒有辦法把 vim 當作 pipe 工具來用，
後來試出的用法如下:


    vim -i NONE -e -X -c ':123 move 117' -c':wq' /usr/bin/genkernel


這一段的語法，是指將第 123 行換到 117 行，然後存檔。
