在本機連線到 OpenShift 上的 MySQL 

[[code]]
// 啓用 port forward (會將 openshift 上的 port 對應到本地)
$rhc port-forward -a applicationName
// 使用 show app 可以看到資料庫連線的帳號密碼 
$ rhc show-app --app applicationName
// 使用 TCP 連線到資料庫 
$ mysql -u USERNAME -p -h 127.0.0.1
[[/code]]
